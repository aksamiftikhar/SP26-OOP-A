# Lecture 11: equals() with ArrayList and Arrays

Run `EqualsCollectionsArraysDemo.java` to demonstrate:

1. `ArrayList.contains`, `indexOf`, and `remove(Object)` behavior with and without overriding `equals()`.
2. `Arrays.equals` for object arrays and how it depends on element `equals()`.

## Run

```powershell
javac src\lecture11_EqualsCollectionsArrays\EqualsCollectionsArraysDemo.java
java -cp src lecture11_EqualsCollectionsArrays.EqualsCollectionsArraysDemo
```
